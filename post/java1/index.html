Java?

Java adalah nama dari sebuah bahasa pemrograman high level. Java dapat digunakan untuk menulis program. Program adalah instruksi atau statement dalam bahasa pemrograman bisa Java, Python, Cobol dan lain-lain yang dimengerti oleh komputer. Dengan adanya program, komputer dapat dioperasikan oleh programmer. Misalnya ketika kita hendak presentasi, pasti kita menggunakan program yang bernama Microsoft Power Point, yang dimana program tersebut dibuat menggunakan bahasa pemrograman. Java dikembangkan oleh Sun Microsystem dan diperkenalkan kepada secara umum pada tahun 1995.Karena Java adalah pemrograman high level maka perintah-perintah yang digunakan mudah dimengerti oleh halayak umum atau universal. Sebagai bahasa universal, Java sering dijumpai di berbagai platform seperti (Windows, Unix, Linux dan Mac).

Perbedaan program dan aplikasi

Program adalah serangkaian instruksi atau statemen yang ditulis oleh programmer untuk melakukan suatu fungsi pada komputer. Aplikasi dapat diartikan sebagai program dalam bentuk perangkat lunak yang dapat dijalankan pada operasi sistem yang berfungsi untuk memudahkan kegiatan manusia. Bisa diartikan bahwa aplikasi adalah terjemahan dari sebuah program yang disimpan dalam ekstensi tertentu, contohnya pada sistem windows dimana hasil terjemahan program disimpan dalam format .exe.

Proses installasi Java

  • Buka peramban dengan memasukan kata kunci JDK

  • Kemudian klik JDK Downloads, untuk melakukan download JDK

  • Pilih versi JDK sesuai spesifikasi komputer x86 atau x64

  • Tunggu proses download hingga muncul JDK Installer pada folder downloads

  • Klik open pada JDK,lalu klik Next

  • Tunggu hingga proses instalasi selesai
